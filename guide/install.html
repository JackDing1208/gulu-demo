# 安装流程

# 环境要求

使用该框架请先自行安装 node.js、npm 和 Vue-cli 项目开发时node版本为10.16

# 安装

npm install jd-vue-ui
or  
yarn add jd-vue-ui

node-sass相关依赖可能导致安装较慢

WARNING

目前该UI框架为练手项目,可能存在BUG,谨慎在生产环境中使用
